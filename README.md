# Used Bike Price Prediction

## Overview

This project predicts the selling price of used bikes based on various features using a Machine Learning regression model. A Flask web application allows users to enter bike details and receive an estimated price instantly.

## Features

- Used bike price prediction
- Machine Learning regression model
- Flask web application
- Clean user interface
- Instant price estimation

## Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS

## Installation

```bash
git clone <repository-url>
cd Bike-Price-Prediction

pip install -r requirements.txt
```

## Run

```bash
python application.py
```

Visit

```
http://127.0.0.1:5050/
```

## Model

The regression model predicts used bike prices based on attributes such as brand, age, kilometers driven, owner information, and other relevant features.
